HTML — The Structure

HTML (HyperText Markup Language) is the skeleton of every webpage. It uses tags — words inside angle brackets — to define what each piece of content is. Is it a heading? A paragraph? A link? A table?

<!-- A heading --> <h1>Atlanta Sports Fan</h1> <!-- A paragraph --> <p>The World Cup is coming to Atlanta!</p> <!-- A link --> <a href="worldcup.html">See the schedule</a> <!-- A button --> <button onclick="doSomething()">Click me!</button>

On this site, HTML defines the navigation bar, the match table, the cards, the hero sections, and everything else you can see on the page.

🎨

CSS — The Style

CSS (Cascading Style Sheets) makes HTML look good. Without CSS, every page would be plain black text on a white background. CSS controls colors, fonts, spacing, layout, animations, and more.

/* CSS Variables — define colors once, reuse everywhere */ :root { --color-red: #a8132a; --color-gold: #f0b429; } /* Style the nav bar */ nav { background-color: var(--color-bg); border-bottom: 2px solid var(--color-red); position: sticky; /* stays at top while scrolling */ top: 0; } /* Animate a button on hover */ .btn:hover { transform: translateY(-2px); /* lift up 2px */ transition: transform 0.15s; }

All the styles for this site live in style.css — one shared file. That's why every page looks consistent. Change the gold color in one place and it updates across the whole site instantly.

⚡

JavaScript — The Behavior

JavaScript makes pages interactive. It can respond to clicks, read input, show/hide content, count down to a date, and much more — all without reloading the page.

// Store all matches in an array of objects const matches = [ { home: "Spain", away: "Cabo Verde", atl: true }, { home: "Morocco", away: "Haiti", atl: true }, { home: "Brazil", away: "France", atl: false }, ]; // Filter to Atlanta games only const atlGames = matches.filter(m => m.atl === true); // Countdown timer — runs every second setInterval(function() { const target = new Date('2026-06-15T12:00:00'); const diff = target - new Date(); const days = Math.floor(diff / 86400000); document.getElementById('cd-days').textContent = days; }, 1000);

On the World Cup page, JavaScript does three big jobs:
① Builds all 104 table rows from the data array
② Powers the filter buttons and search box
③ Runs the live countdown timer


Your Challenges

Things to build next — in order of difficulty.

  • Change the color scheme Open style.css and change --color-red and --color-gold to new colors. See how the whole site changes!
  • Add a new sport card on the home page Copy one of the .sport-card blocks in index.html and change the emoji, name, and description.
  • Add a real Braves or Falcons game to sports.html Look up a real upcoming game and replace the "Coming Soon" placeholder with a real date, opponent, and time.
  • Add a "Favorite Teams" feature with JavaScript Add a ⭐ button to each team card. When clicked, save the choice with localStorage and show it on the home page.
  • Make the site mobile-friendly Resize your browser window to phone size. What breaks? Use CSS media queries to fix it.
  • Add a dark/light mode toggle Add a 🌙/☀️ button to the nav bar. When clicked, use JavaScript to switch a CSS class on <body> that changes the colors.
  • Deploy to GitHub Pages Create a free GitHub account, upload this folder, and turn on GitHub Pages. Your site goes live on the internet!
