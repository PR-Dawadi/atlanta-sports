# 🏙️ Atlanta Sports Fan

A website about Atlanta sports, live events, and FIFA World Cup 2026 — built with plain HTML, CSS, and JavaScript. No frameworks, no libraries, no build tools.

Built as a learning project to teach web development from the ground up.

---

## 📄 Pages

| File | Description |
|------|-------------|
| `index.html` | Home page — links to all sections |
| `worldcup.html` | ⚽ Full FIFA World Cup 2026 schedule (all 104 matches) |
| `sports.html` | Atlanta teams — Falcons, Braves, Hawks, Atlanta United |
| `concerts.html` | Concerts & live events in Atlanta |
| `learncode.html` | Behind-the-scenes: how this site was built |
| `style.css` | Shared stylesheet used by all pages |

## 🚀 Running Locally

1. Clone the repo:
   ```
   git clone https://github.com/YOUR_USERNAME/atlanta-sports.git
   ```
2. Open the folder in [VS Code](https://code.visualstudio.com/)
3. Install the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension
4. Right-click `index.html` → **Open with Live Server**

That's it — no installs, no terminal commands needed.

## 📚 Learning Goals

This project is designed to teach:
- How HTML, CSS, and JavaScript work together
- CSS variables and a shared stylesheet across multiple pages
- JavaScript arrays and objects to store and display data
- DOM manipulation — building HTML from data with JS
- Filter and search functionality without any library
- `setInterval()` for a live countdown timer
- Git & GitHub workflow
- Deploying a static site with GitHub Pages
